## 名词的性和数
俄语名词分为阳、阴、中性。在各种变化中复数群体的状态比较特殊
### 性
判断一个名词的性，主要看词尾（此处仅写通常情况）

| 阳性       | 词尾  | 阴性       | 词尾  | 中性          | 词尾  |
| :------- | :-- | :------- | :-- | :---------- | :-- |
| брат     | 硬辅音 | сестра́  | -а  | окно́       | -о  |
| слова́рь | -ь  | пе́сня   | -я  | мо́ре       | -е  |
| музе́й   | -й  | тетра́дь | -ь  | упражне́ние | -ие |
### 数
主格变复数遵循的主要规则
- 基础规则：阳性加 -ы，阴性 -а → -ы，中性 -о → -а，-е → -я，-ие → -ия
- **例外**：г, ж, к, х, ч, ш, щ, 软音符号后-а → -и
- -й/-я → -и
- 以 **-ь** 结尾的**阴性**名词，复数也是 **-и**

> [!NOTE] 总结
> 除了阳性辅音，阴性的 -а加-ы和中性的变化不一样，其他都是变-и（短）

## 人称代词
### 主格基本形式
| **人称** | **单数** | **性** | **复数** |
|---------|---------|--------|---------|
| 第一人称 | **я** (我) | - | **мы** (我们) |
| 第二人称 | **ты** (你) | - | **вы** (你们/您) |
| 第三人称 | **он** (他) | 阳性 | **они** (他们/她们/它们) |
| | **она** (她) | 阴性 | |
| | **оно** (它) | 中性 | 
### 人称代词的变格

| **主格** | **第四格** | **第六格**  | **前置词后形式**    |
| ------ | ------- | -------- | ------------- |
| я      | меня    | (о) мне  | -             |
| ты     | тебя    | (о) тебе | -             |
| он     | его     | (о) нём  | **него, нём** |
| она    | её      | (о) ней  | **неё, ней**  |
| оно    | его     | (о) нём  | **него, нём** |
| мы     | нас     | (о) нас  | -             |
| вы     | вас     | (о) вас  | -             |
| они    | их      | (о) них  | **них**       |
第三人称代词（он, она, оно, они）在**带前置词**的第四格和第六格时，词首要加 **н-**
**第四格** = 动作的**直接对象**，动词的直接宾语
**第六格** = **总是带前置词** о（关于）/в（在）/на（在...上）
**第二格**

| 人称                   | 二格/四格                    | 是否同形           |     |
| -------------------- | ------------------------ | -------------- | --- |
| **я, ты, мы, вы**    | меня/тебя/нас/вас        | **同形**         |     |
| **он/оно, она, они** | него/неё/них - его/её/их | **不同形，二格加 н-** |     |

二格的功能
**1. 表示"拥有"** —— 用在 **у** 后面  
**у них** 他们有  
**у неё** 她有  
**у меня**  我有
**2. 表示"没有"** —— 用在 **нет** 后面  
**нет времени** 没有时间  
**нет неё** 她不在（这儿）
## 物主代词
### 主格基本形式

|         | 阳性   | 阴性    | 中性    | 复数    |
| :------ | :--- | :---- | :---- | :---- |
| **我的**  | мой  | моя́  | моё   | мои́  |
| **你的**  | твой | твоя́ | твоё  | твои́ |
| **我们的** | наш  | на́ша | на́ше | на́ши |
| **你们的** | ваш  | ва́ша | ва́ше | ва́ши |
| **它的**  | его  | ~     | ~     | ~     |
| **她的**  | её   | ~     | ~     | ~     |
| **他的**  | его  | ~     | ~     | ~     |
| **他们的** | их   | ~     | ~     | ~     |
### 物主代词的变格
| 代词       | 格   | 阳性                 | 阴性     | 中性     | 复数             |
| -------- | --- | ------------------ | ------ | ------ | -------------- |
| **我的**   | 四格  | мой / моего́       | мою́   | моё    | мои́ / мои́х   |
|          | 六格  | моём               | мое́й  | моём   | мои́х          |
| **你的**   | 四格  | твой / твоего́     | твою́  | твоё   | твои́ / твои́х |
|          | 六格  | твоём              | твое́й | твоём  | твои́х         |
| **我们的**  | 四格  | наш / на́шего      | на́шу  | на́ше  | на́ши / на́ших |
|          | 六格  | на́шем             | на́шей | на́шем | на́ших         |
| **你们的**  | 四格  | ваш / ва́шего      | ва́шу  | ва́ше  | ва́ши / ва́ших |
|          | 六格  | ва́шем             | ва́шей | ва́шем | ва́ших         |
| 他/她/它/他们 | —   | **его/её/их 永远不变** |        |        |                |
**四格** = 直接宾语  
例：вижу **мою́** книгу, люблю́ **твоего́** друга

**六格** = 前置词 в/на/о + 六格  
例：в **мое́й** книге, о **твоём** друге

**注意**：阳性、复数四格中，**有生命的**用二格形式（如 моего́, твоих）

## 形容词变化
### 性的变化
- **阳性**：硬 -ый，软/唏音后 -ий（如：хоро́ший чай）
- **阴性**：硬 -ая，软 -яя
- **中性**：硬 -ое，软 -ее（取决于名词词尾前是硬辅音还是软辅音/唏音）
- **复数**： 硬 → -ые，软 → -ие

> [!NOTE] 区分硬变化和软变化
> **硬变化** → 词干以**硬辅音**结尾
> 
> - 硬辅音：除ж, ч, ш, щ, й, ь 之外的所有辅音
> - **词尾**：-ый, -ая, -ое, -ые
> 
> **例子**：
> 
> - **нов**ый → 词干 **нов-** (в是硬辅音)
> - **интересн**ый → 词干 **интересн-** (н是硬辅音)
> 
> **软变化** → 词干以**软辅音**结尾
> 
> - 软辅音：**й**, **ь**, 或 **唏音** (ж, ч, ш, щ)
> - **词尾**：-ий, -яя, -ее, -ие
> 
> **例子**：
>- синий → 词干 синь
>- крайний → 词干 крайнь

#### A. 阳性与中性的“连体婴”关系

除了**主格**（和宾格）不同，阳性和中性的形容词在**属格、与格、工具格、前置格**的词尾是**完全一样**的。

- _例子：Genitive 都是 -ого，Dative 都是 -ому。_
    
#### B. 阴性的“坚强外壳”

阴性形容词的变化非常“偷懒”。在**属格、与格、工具格、前置格**这四个格中，词尾基本上都是 **-ой**。

- _注意：工具格偶尔可见古体写法 -ою，但现代俄语常用 -ой，所以这就意味着阴性单词除了主格(-ая)和宾格(-ую)外，其他情况大概率都读“oy”。_


#### C. 宾格的“有生命”规则 (Animacy Rule)

这是唯一的“陷阱”，请务必记住：

- **适用范围：** 阳性单数、任何性的复数。
    
- **规则：**
    
    - 如果是**死物**（无生命，如桌子）：宾格 = **主格**。
        
    - 如果是**活物**（有生命，如人、猫）：宾格 = **属格**。
        
- _阴性单数不受此影响，宾格永远是 -ую。_

### 3. 软变化与拼写规则 (Soft Stem & Spelling Rules)

如果形容词词干是以软音结尾（如 _синий_ 蓝色的），或者遇到特定的辅音（г, к, х, ж, ш, ч, щ），你需要应用**元音互换**。不要死记硬背新表格，只需将硬词尾的元音替换一下：

|**硬元音 (Hard)**|**对应软元音 (Soft)**|**备注**|
|---|---|---|
|**ы**|**и**|遇到 **г, к, х, ж, ш, ч, щ** 这7个字母后，必须写 **и** (不能写 ы)。<br><br>  <br><br>_例：русск**и**й (不是 русскый)_|
|**о**|**е**|在重音不在词尾时，唏音(ж, ш, ч, щ, ц)后写 **е**。<br><br>  <br><br>_例：хорóш**е**го (不是 хорошого)_|
|**а**|**я**|软变化阴性主格。<br><br>  <br><br>_例：син**я**я_|
|**у**|**ю**|软变化阴性宾格。<br><br>  <br><br>_例：син**ю**ю_|

**应用示例（软变化 синий vs 硬变化 новый）：**

- **属格 (Gen):** нов**ого** $\rightarrow$ син**его** (o变e)
    
- **工具格 (Inst):** нов**ым** $\rightarrow$ син**им** (ы变и)
    
- **阴性主格:** нов**ая** $\rightarrow$ син**я**я (а变я)

### 4. 极简速查口诀

> 阳中属格 og(v)o，与格 omu；
> 
> 阴性四格皆 oy，宾格 uyu。
> 
> 工具 ym / ymi，复数 yx 记心头。

_(注：ogo 发音为 ovo)_
